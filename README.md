# Security-Notes

A personal knowledge base for cybersecurity studies, notes, concepts, methodologies, and practical references.

This repository serves as a centralized collection of security-related topics gathered throughout my learning journey, including networking, operating systems, offensive security, defensive security, scripting, and certification preparation.

---

## Purpose

The goal of this repository is to:

- Document cybersecurity concepts and methodologies
- Organize study notes in a structured format
- Create a quick-reference resource for future use
- Track learning progress over time
- Build a public knowledge portfolio
